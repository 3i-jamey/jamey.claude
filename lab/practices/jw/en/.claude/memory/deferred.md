# Deferred Items

Items postponed from past sessions. The `/new-phase` skill re-evaluates these each run — matching items are flagged and pulled into scope.

Two categories:
- **[DECISION]** — Session idea not yet committed to the Spec. Needs a Spec write before it can become a phase step.
- **[FEATURE]** — Already specced in the Spec, just not implemented yet. Can go straight into a phase.
